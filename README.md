# tic-tac-toe-optimal-move

Простая и быстрая JavaScript-библиотека для выбора **оптимального хода** в крестиках-ноликах (3×3).

- Выигрывает, если может
- Блокирует противника
- Берёт центр → углы → стороны

Идеально подходит для игр — непроигрывающий ИИ.

## Установка

```bash
npm install tic-tac-toe-optimal-move
```

## Как пользоваться

```js
const { getOptimalTurn } = require("tic-tac-toe-optimal-move");

// Поле — массив из 9 чисел (индексы 0–8, слева направо, сверху вниз)
// -1 = пустая клетка
//  0 = O (нолики)
//  1 = X (крестики)

const field = [0, 0, -1, -1, 1, 1, 0, -1, -1];

// Ход за O (0)
const move = getOptimalTurn(field, 0);
console.log(move); // → 2 (немедленная победа)
```
